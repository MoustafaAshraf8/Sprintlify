# Sprintlify

### 1. Push database migrations

```
npm run supabase:migrate-local
```

### 2. Generate database types

```
npm run supabase:update-types-local
npm run drizzle:introspect-local
```

### 3. Run development server

```
npm run dev
```

### 4. Access your server at:

`http://127.0.0.1:8787`
